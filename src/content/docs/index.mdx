# Welcome to CodeVault

CodeVault is an enterprise-ready, self-hosted AI coding assistant platform. It provides GPU-accelerated code completion and chat capabilities with enterprise features like multi-tenancy, RBAC, audit logging, and observability.

## Why CodeVault?

- **100% Private** - Your code never leaves your infrastructure
- **GPU Accelerated** - Leverage NVIDIA CUDA for fast inference
- **OpenAI Compatible** - Drop-in replacement for existing tools
- **Enterprise Ready** - RBAC, audit logs, multi-tenancy
